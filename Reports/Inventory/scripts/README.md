# Scripts

- `package-report.ps1` — zip `Inventory Report/` (defaults to this module’s PBIP folder)
- `validate-structure.ps1` — portfolio contract check for `Inventory`
- `archive-prune.ps1` — trim `Exports/Server Packages/archive`
- `clear-model-cache.ps1` — remove `.pbi/cache.abf` for blank-on-open

`clear-cache-and-open.ps1` at module root is a convenience shortcut (clear + launch Desktop).

# Chat context (Cursor)

## Where to put files

Save screenshots here when chat image upload fails:

**`Shared/ChatContext/images/`**

Use normal names; the assistant finds **what you added most recently** by file date, not by `@` mentions.

## How to ask

Say things like **“see the last screenshot”**, **“last two screenshots”**, or **“what’s in my latest screen capture?”** The assistant should list this folder, sort image files by **last modified time** (newest first), open the newest one(s), and respond from that.

Supported extensions: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`.

**Learn for next time:** when you want mistakes from a capture remembered, say so (e.g. “learn from this”). The assistant appends concise bullets to **`LESSONS.md`** and, for Finance-report issues, mirrors into **`Reports/Finance/Project Memory/DECISIONS.md`** (or **`MODEL_NOTES.md`** when appropriate).

This folder is not portfolio source of truth—delete old captures when you like.

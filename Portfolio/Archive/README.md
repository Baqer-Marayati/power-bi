# Archive

This folder stores portfolio-level historical material that should no longer be treated as active work.

## Rules

- Archive by category and meaning.
- Prefer self-describing names with date, subject, and status.
- Add a short README inside major archived folders when context would otherwise be lost.

Examples:
- `2026-03-22_finance-shell-v1_superseded`
- `2026-03-22_old-benchmark-set_historical`
- `2026-03-22_logo-overlay-experiment_failed`
